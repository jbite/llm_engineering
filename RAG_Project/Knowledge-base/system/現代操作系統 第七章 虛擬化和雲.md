## 7.2 虛擬化的必要條件

對於虛擬機來講，非常重要的一點是要像真實的機器那樣運轉。例如，虛擬機要能像真實的機器那樣啟動，支持安裝任意操作系統。

虛擬機管理程式需要在以下三個維度上有良好的表現：

1. 安全性：虛靜機管理程式應完全掌握虛擬機資源
2. 保真性：程式在虛擬機上執行的行為應與在實機上相同
3. 高效性：虛擬機中運行的大部分代碼應不受虛擬機管理程式的干涉。
